מבוא לכתובות IP - המדריך הבסיסי
דף ה HTML  הזה נבנה כדי לעשות סדר וליצור היכרות ראשונית עם המושג כתובת IP  איך היא עובדת, מה ההבדל בין הסוגים השונים, ולמה בכלל צריך אותה בעולם האינטרנט.
מה תמצאו בדף הזה?  מה זה בכלל  IP -  IPv4 מול IPv6  - למה נגמרו לנו הכתובות הישנות ואיך החדשות מצילות את המצב.   כתובת ציבורית (Public) מול פרטית (Private) מה העולם רואה ומה קורה בתוך הנתב הביתי שלכם.
 טכנולוגיות שנעשה בהן שימוש - HTML5 לבניית המבנה והתוכן בצורה נגישה ונקייה.
הערה: הפרויקט נבנה למטרות למידה ותרגול בסיסי של מושגי רשתות ופיתוח פרונט-אנד. מוזמנים לחקור את הקוד ולשנות אותו בכיף!
